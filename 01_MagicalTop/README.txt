#	作品概要
#-----------------------------------------
◎情報
　タイトル：MagicalTop
　ジャンル：3Dアクションシューティングゲーム
　開発環境：DirectX9
　使用言語：C++
　開発期間：2023年5月下旬～9月下旬
　GitLink ：https://github.com/YuichiFujita/MagicalTop

◎アピールポイント
・ショップ機能による自由なプレイスタイルの実現
　敵から出る経験値を集めることで、レベルアップ！
　レベルを使用することで、自身の能力を強化できます。
　強化内容はランダム且つ様々。自分なりの強化を見つけられます。

・起伏による臨場感の増加
　地形に起伏をつけることで、リアルな戦場感を与えています。
　起伏の生成をエディター化することで効率化をしています。
